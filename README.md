# Configuración del proyecto

## Requisitos previos

Antes de compilar o ejecutar el proyecto, es necesario configurar el archivo de propiedades locales con las API keys correspondientes.

## Pasos de configuración

1. En la raíz del proyecto, copiá y renombrá el archivo de ejemplo:

```bash
cp local.properties.example local.properties
```

2. Las API keys necesarias ya fueron provistas a través de la entrega en Moodle. Abrí el archivo `local.properties` y completá los valores correspondientes con las keys recibidas.

> **Nota:** El archivo `local.properties` está ignorado por Git (`.gitignore`) y no debe subirse al repositorio bajo ninguna circunstancia, ya que contiene información sensible.
